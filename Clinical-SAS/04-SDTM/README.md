# SDTM (Study Data Tabulation Model)

This folder contains SDTM domain mapping practice based on CDISC standards.

## Domains Covered

- DM (Demographics)
- AE (Adverse Events)
- LB (Laboratory)
- CM (Concomitant Medications)
- MH (Medical History)
- EX (Exposure)
- VS (Vital Signs)

- dm.sas
- ae.sas
- lb.sas
